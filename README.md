# donaldfountain.ai

Personal blog on AI governance, explainability, and systems engineering.

Built with [Hugo](https://gohugo.io/) and the [Stack theme](https://github.com/CaiJimmy/hugo-theme-stack).

Hosted on [Azure Static Web Apps](https://azure.microsoft.com/en-us/services/app-service/static/).

## Local Development

```bash
cd hugo-site
hugo server -D
```

Visit <http://localhost:1313>

## Content

Topics include:

- AI policy analysis and governance implementation
- Explainable AI (xAI) technical implementations
- Systems engineering for responsible AI
- Radio astronomy data analysis applications
